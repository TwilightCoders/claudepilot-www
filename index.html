Features

Everything you need to stay connected.

Built from the ground up for developers who run Claude Code on remote machines.

Native Terminal

Real xterm emulation powered by SwiftTerm. Full color, scrollback, and VT100 — not a web view.

tmux Sessions

Independent viewport with grouped sessions. Your phone gets its own window size, no fighting with your desktop.

Quick Actions

One-tap approve, reject, continue, or interrupt. No typing on a tiny keyboard when a button will do.

Reach Any Machine

ProxyJump, SOCKS5, and WireGuard connectivity. Reach your dev box through Tailscale, firewalls, or VPNs.

Auto-Reconnect

Seamless recovery on network drops. Switch from WiFi to cellular and back without losing your session.

Secure by Default

SSH keys stored in the iOS Keychain. Ed25519, RSA, and ECDSA — with passphrase support.

How It Works

Up and running in three steps.

No account needed. No server component. Just SSH.

1

Install the CLI

The companion CLI sets up tmux session management on your dev machine.

brew install claudepilot
2

Add Your Host

Configure your SSH connection in the app — hostname, key, and network provider.

3

Tap In

Connect to any tmux session from your phone. Watch Claude work, approve changes on the go.

Install

Get the companion CLI.

Install on any machine you want to connect to from your phone.

curl -fsSL https://claudepilot.net/install.sh | bash

Sets up the claudepilot CLI for tmux session management and discovery.
Or install via Homebrew: brew install claudepilot
